# Structure

A standard theme's structure will look like this:

```text
├───mocks/
│   └───index.json
└───theme/
    ├────assets/
    │   └───base.css
    ├────blocks/
    │   └───cta.hbs
    ├────partials/
    │   └───layout.hbs
    └────templates/
        └───basic-page.hbs
```
